# Engineering Management System

Professional management system for engineers working in:

* Building & Construction
* Mining Operations
* Metal Fabrication
* Technical Services

## Main Features

✅ Secure Login System
✅ Role Based Access
✅ Admin Dashboard
✅ Project Management
✅ Staff Management
✅ Reports Upload
✅ Daily Progress Tracking
✅ Material Records
✅ Mining Operations Tracking

## User Roles

* Admin
* Project Manager
* Site Manager
* Staff
* Managing Director

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* XAMPP
